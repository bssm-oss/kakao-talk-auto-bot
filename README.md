# 🤖 Chatbotchichi (챗봇치치)

안드로이드용 카카오톡 봇 엔진 프로젝트입니다. **메신저봇R**의 핵심 기능을 벤치마킹하여, 앱 내에서 직접 자바스크립트 코드를 작성하고 실제 카카오톡 알림에 반응하는 봇을 운영할 수 있습니다.

## 🌟 주요 기능

- **멀티 봇 관리**: 여러 개의 `.js` 스크립트를 생성하고 개별적으로 ON/OFF 할 수 있습니다.
- **실시간 코드 에디터**: PC 연결 없이 스마트폰에서 바로 봇 로직을 수정하고 저장할 수 있습니다.
- **디버깅 룸 (시뮬레이터)**: 실제 카톡방과 유사한 UI에서 봇의 반응을 즉시 테스트할 수 있습니다.
- **세션 캐싱 (SessionManager)**: 알림을 한 번만 받으면, 해당 채팅방의 답장 세션을 저장하여 이후에 지속적으로 메시지를 보낼 수 있습니다.
- **폴링 봇 마법사**: 코드 작성 없이 Webhook URL과 주기만 입력하여 외부 서버(N8N 등)의 데이터를 카톡으로 전송합니다.
- **N8N 연동**: `executeWorkflow` 함수를 통해 봇이 수신한 메시지를 외부 자동화 툴로 즉시 전송할 수 있습니다.

## 🛠 기술 스택

- **언어**: Kotlin 2.0+
- **엔진**: [Rhino-Android](https://github.com/Faendir/rhino-android) (JavaScript ES5/ES6 지원)
- **네트워크**: OkHttp3 (외부 API 및 웹훅 통신용)
- **비동기 처리**: Kotlin Coroutines
- **UI**: ConstraintLayout, Material Design

## 🚀 시작하기

1. **빌드**: Android Studio에서 프로젝트를 빌드하여 기기에 설치합니다.
2. **권한 설정**: 앱 실행 후 상단의 **[권한 설정]** 버튼을 눌러 `Chatbotchichi` 앱의 알림 접근을 허용합니다.
3. **봇 생성**: '+ 생성' 또는 '폴링 봇' 버튼을 눌러 첫 번째 봇을 만듭니다.
4. **테스트**: '디버깅 룸 입장' 후 메시지를 입력하여 봇이 정상 작동하는지 확인합니다.
5. **실전**: 실제 카카오톡 방의 알림이 켜져 있는지 확인하면 봇이 작동을 시작합니다.

## ⚠️ 주의사항

- **알림 설정**: 카카오톡 알림 내용이 '이름 + 메시지'로 설정되어 있어야 내용을 읽을 수 있습니다.
- **배터리**: 앱이 백그라운드에서 계속 실행되어야 하므로 '배터리 최적화 제외' 설정을 권장합니다.
