# 카톡 자동답장 도우미

카카오톡 알림을 로컬에서 수집하고, 선택한 방에만 **로컬 LLM(llama.cpp)** 기반 AI 자동응답을 연결하는 안드로이드 앱입니다. 모든 데이터를 기기 안에 저장하고, 인터넷 없이도 AI 답장이 작동합니다.

## 한눈에 보기

| 항목 | 내용 |
| --- | --- |
| 응답 대상 | 선택한 방 또는 모든 방 (토글 가능) |
| AI 엔진 | llama.cpp 기반 로컬 LLM (GGUF 모델) |
| 추천 모델 | Qwen2.5 1.5B (한국어 우수, ~1.1GB) |
| 저장 방식 | 대화/설정/로그를 기기 로컬에 저장 |
| 응답 조건 | AI 판단, 호출어/멘션, 질문/명령, 키워드, 고정 답장 |
| 운영 제어 | OFF 상태에서도 메시지 저장 유지, 답장만 중지 |
| UI | 라이트/다크 모드, 키보드 자동 스크롤 |

## 핵심 기능

- **로컬 LLM 답장**: llama.cpp + GGUF 모델로 인터넷 없이 AI 답장 생성
- **모델 다운로드**: 앱 안에서 Qwen2.5 1.5B 모델 다운로드 (~1.1GB)
- **카카오톡 알림 기반**: 알림 리스너로 메시지를 감지하고 PendingIntent로 답장
- **방별 메모**: 각 채팅방별 컨텍스트를 메모로 저장, AI가 답변 시 참고
- **모든 방 모드**: 토글 하나로 모든 채팅방에서 AI 답장 활성화
- **페르소나 설정**: 내 이름, 말투, 답변 스타일을 커스터마이징
- **자동 대화 요약**: 최근 대화를 자동으로 분석해 메모에 반영
- **트리거 조건**: AI 판단, 멘션, 질문, 키워드 등 다양한 반응 조건

## 지원 범위

- 지원: llama.cpp 기반 로컬 LLM (GGUF 모델, CPU 동작)
- 지원: Qwen2.5, Llama, Gemma 등 GGUF 호환 모델
- 지원: AI 판단형 응답, 멘션 반응, 질문/명령형 반응, 고정 답장
- 미지원: OpenAI / Claude / Gemini 등 원격 API 호출
- 미지원: GPU 가속 (현재 CPU 전용)

## 빠른 시작

1. 앱 설치 후 알림 접근 권한을 허용합니다.
2. `AI 답장 설정`에서 내 이름과 페르소나를 저장합니다.
3. **모델 다운로드** 버튼을 눌러 LLM 모델을 다운로드합니다 (~1.1GB).
4. `대상 방 관리`에서 자동응답할 방을 추가하거나, `모든 방에서 답장`을 켭니다.
5. 방별 `메모`에서 AI가 참고할 컨텍스트를 적습니다.
6. 메인 화면에서 `AI 자동 답장`을 켭니다.

## 모델

앱은 기본적으로 **Qwen2.5-1.5B-Instruct-Q4_K_M** GGUF 모델을 사용합니다.

| 모델 | 크기 | 한국어 | 비고 |
| --- | --- | --- | --- |
| Qwen2.5 1.5B Q4 | ~1.1GB | 우수 | 기본 추천 |
| Qwen2.5 3B Q4 | ~2GB | 매우 우수 | 고용량 기기 권장 |
| Gemma 2 2B | ~1.5GB | 양호 | Google 모델 |

모델은 앱 내 다운로드 기능으로 받을 수 있으며, 다른 GGUF 모델도 직접 추가할 수 있습니다.

## 성능 가이드

- 1.5B 모델: 답장 생성 1~3초 (최근 기준)
- 3B 모델: 답장 생성 3~6초
- 최소 2GB RAM 권장
- 첫 모델 로딩 시 5~10초 소요

## 개발

```bash
./gradlew testDebugUnitTest lintDebug assembleDebug assembleRelease
```

- NDK 27 이상 필요 (llama.cpp 네이티브 빌드용)
- CMake 3.22 이상 필요
- llama.cpp는 빌드 시 자동으로 다운로드됩니다

## CI / 릴리즈

- `Android CI`: JVM 테스트, lint, debug/release APK 빌드
- `Release APK`: `v*` 태그 푸시 시 서명된 release APK를 GitHub Release에 첨부
