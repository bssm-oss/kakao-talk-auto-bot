# 행동 강령

이 프로젝트는 모든 기여자와 사용자에게 안전하고 존중받는 협업 환경을 제공하는 것을 목표로 합니다.

## 우리가 기대하는 태도

- 상대를 존중하고, 비판보다 개선 제안을 우선합니다.
- 기술적 논의는 근거와 재현 가능한 정보에 기반합니다.
- 다양한 배경, 경험, 관점을 존중합니다.
- 버그 제보, 리뷰, 문서 기여도 코드 기여만큼 중요하게 다룹니다.

## 허용되지 않는 행동

- 인신공격, 조롱, 비하, 혐오 표현
- 반복적인 트롤링, 논의 방해, 악의적 스팸
- 허가 없는 개인정보 공개
- 보안 취약점을 공개 이슈로 바로 게시하는 행위

## 운영 원칙

- 유지보수자는 커뮤니티 안전을 위해 경고, 코멘트 제한, 참여 제한 등의 조치를 취할 수 있습니다.
- 논의는 가능한 한 공개적으로 하되, 보안 이슈는 `SECURITY.md` 절차를 따릅니다.
- 리뷰와 PR 논쟁은 “작게 나누고, 재현하고, 문서화한다”는 원칙으로 정리합니다.

## 신고

- 공개 스레드에서 해결하기 어려운 문제는 유지보수자에게 별도 전달합니다.
- 보안 관련 문제는 공개 이슈 대신 `SECURITY.md` 절차를 사용합니다.
