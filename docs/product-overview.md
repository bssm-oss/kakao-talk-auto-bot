# 제품 개요

`kakao-talk-auto-bot` 은 카카오톡 알림을 읽어 특정 방에만 AI 자동응답을 붙이는 안드로이드 앱입니다.

핵심은 다음과 같습니다.

1. **로컬 저장**: 대화 흐름과 설정을 기기 안에 저장
2. **방 선택**: 모든 방이 아니라 선택한 방만 응답 대상
3. **선택적 응답**: 멘션/질문/고정 규칙/AI 판단에 따라 필요한 순간만 답장
4. **간단한 UX**: 권한 → 응답 설정(Gemma 4 로컬) → 방 추가 → CSV 가져오기 → ON/OFF
5. **직접 제어 가능한 UI**: 메인 화면에서 상태, 로그, 테마 모드를 바로 관리
6. **자동 메모리**: 최근 대화와 링크 공유 패턴을 바탕으로 방 메모리/참여자 성향을 자동 보강
7. **자동 페르소나 힌트**: 최근 내 발화 기준으로 답장 말투를 자동 보정

현재 기본 제품 경로는 **Gemma 4 로컬 답장 엔진**입니다.
저장된 예전 OpenAI 설정이 있어도 런타임에서는 자동으로 로컬 Gemma 경로로 정리됩니다.
소비자 계정 로그인 재사용, OAuth 패스스루, 원격 OpenAI API 경로는 지원하지 않습니다.
