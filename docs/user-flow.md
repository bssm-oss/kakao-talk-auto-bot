# 사용자 플로우

1. 메인 화면에서 알림 접근 권한 허용
2. 응답 설정 화면에서
   - 내 이름
   - 페르소나
   - Gemma 4 로컬 엔진 상태 확인
   - 기본 트리거 성향
   - 최근 내 발화 기준 자동 페르소나 힌트 적용
   를 저장
3. 대상 방 관리 화면에서 자동응답할 방 추가
4. 방 메모리 화면에서
   - 방 메모리
   - 자동 생성되는 방 메모리 요약 확인
   - 허용/차단 발화자
   - 고정 답장 목록
   - 트리거 조건
   을 저장
5. CSV 이력을 방에 가져와 과거 맥락 보강
6. 메인 화면에서 AI 자동 답장 ON/OFF 제어
7. 메인 화면에서 시스템/라이트/다크 테마 전환

현재 메인 화면, 응답 설정 화면, 대상 방 관리 화면, 방 메모리 화면은 같은 플로우를 유지한 채 평평한 카드, 절제된 강조색, 정돈된 간격과 버튼 계층으로 시각 정리를 적용합니다. 응답 설정 화면은 로컬 Gemma 엔진 상태와 그라운딩 기준을 바로 확인할 수 있습니다.

로컬 엔진은 현재 메시지, 최근 방 이력, 방 메모리, 자동 메모리 요약, CSV로 가져온 과거 맥락을 함께 보고 짧은 한국어 답장을 만들거나, 확신이 낮으면 답장을 생략합니다.

OFF 상태에서는 답장만 멈추고, 메시지 수집과 세션 갱신은 유지합니다.
